Built for template work, not general-purpose drawing.

Roblox Classic templates have a specific 585×559 grid: torso, arms, legs, waist, each with exact regions. This editor knows that grid by heart.

Compositing

Layer-based editing

Drag to reorder, duplicate, hide, or delete. Every layer keeps its own transform and adjustment state. On-canvas handles for move / scale / rotate. Scroll-wheel scales. Arrow keys nudge.

Adjustments

Non-destructive image adjustments

Brightness, contrast, saturation, hue rotate, plus per-channel RGB shift. Every slider renders in real time. Baked into the exported PNG. Reset in one click.

Roblox-specific

Exact R15 grid awareness

Precise torso, arm, leg, and waist regions. Mode toggle between Shirt and Pants templates. Live mask overlay so you can see what will actually show on the model. Placement presets for Fit Front, F+B, Stretch, Allover, Scatter.

Assets

Asset library browser

Point at a folder, get a thumbnail grid. Drag to canvas and the image lands where you drop it. Click for the default torso spot. Right-click for full-bleed scale.

Word art

15 fonts, six styles

Solid, gradient, neon, shadow, outline, retro. Live preview, adjustable stroke and angle. Add to canvas at any size. Full text wraps, multi-line support.

Clothing details

Auto-generate seams + trim

V-neck, crew, scoop, henley collars. Cuffs, hems, side seams, pockets, plackets, shoulder stripes, arm bands, back yokes. Adjustable width, color, and optional stitched-dash look.

Privacy

No network, no accounts

The app makes zero API calls. Images load from disk. Projects save to disk. Templates download to disk. No telemetry, no analytics, no cloud sync.

Workflow

Save work, download template

Save your workspace as a .r15proj file and reopen it later with every layer and adjustment preserved. Export the finished 585×559 PNG when you're ready to upload to Roblox.

Install.

Windows 10 / 11. Microsoft Store submission pending — for now, install via the NSIS installer or build from source.

1

Download RBX15 Classic Shirt and Pants Maker Setup.exe from Releases.

2

Click through the SmartScreen "unknown publisher" warning. Code signing is deferred until Microsoft Store approval clears.

3

Run the installer. Pick an install directory. Desktop and Start Menu shortcuts are created automatically.

Or build from source

git clone https://github.com/estevanhernandez-stack-ed/RBX15-Shirt-and-Pants.git
cd RBX15-Shirt-and-Pants
npm install
npm run build    # produces dist/ with the installer

Node.js 18+ and Electron 41 required. npm start for dev mode.
